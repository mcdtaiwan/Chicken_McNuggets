# 關於麥克鷄塊體

你知道嗎？當麥克鷄塊與它的靈魂伴侶 — 糖醋醬相遇，  
不僅能釋放出令人無法抗拒的吸引力，還能用來傳遞...秘密訊息！   
  
麥當勞隆重推出「麥克鷄塊體」，26 個英文字母完整呈現！  
想說的、不敢說的，就讓麥克鷄塊為你傳情達意吧。       
※ 字型僅支援部分繪圖軟體與作業系統，另附PNG檔可下載自由拼湊。
